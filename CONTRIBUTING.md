## Quick Start

1. **Install Rust**: Ensure you have the latest stable Rust toolchain
2. **Clone**: `git clone https://github.com/nixval/declarch.git`
3. **Test**: `cargo test --all`
4. **Format**: `cargo fmt`
5. **Lint**: `cargo clippy --all-targets --all-features`
